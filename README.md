# Packages

nix packages

## Usage

to update the packages in this repo do

```shell
$ nix develop
$ just
```

to use them just to

```shell
$ nix run github:StarrFox/packages#<package name>
```

## Binary Cache

there is a binary cache at <https://starrfox.cachix.org>
